Apa itu Web GIS UMKM Desa? +

Web GIS UMKM Desa adalah platform interaktif yang dibuat untuk mendukung dan mempromosikan Usaha Mikro Kecil Menengah (UMKM) di desa. Ini memanfaatkan teknologi Sistem Informasi Geografis (SIG). Peta interaktif dan data geografis dimanfaatkan untuk memudahkan masyarakat, pemerintah, dan investor dalam menemukan serta memahami potensi UMKM di Desa Sukorejo.

Mengapa Web GIS UMKM Penting? +

Untuk mempromosikan UMKM, transparasi data, pengambilan keputusan, dan akses informasi oleh masyarakat dan calon investor terhadap UMKM.

Cara menggunakan Web GIS UMKM Desa Sukorejo +

Memanfaatkan peta interaktif untuk mengeksplorasi lokasi UMKM di Desa Sukorejo. Pergi ke bagian Hasil Web Map untuk mengakses web map UMKM. Untuk mengetahui UMKM yang dipetakan dalam web map, bisa menuju bagian UMKM di Desa Sukorejo pada Menu. Anda bisa mencari tahu UMKM rekomendasi sesuai jenis UMKM yang diinginkan, menggunakan fungsi bagian Rekomendasi UMKM.

Penjelasan Web Map

Web map adalah peta yang bisa diakses menggunakan internet dan dihubungkan menggunakan browser web. Web map tidak hanya peta cetak yang didigitalkan, melainkan peta yang memungkinkan kita sebagai pengguna untuk melihat, berinteraksi, dan menganalisis data geografis secara online. Web map ini bersifat interaktif.

Apa saja fitur yang disediakan dalam web map UMKM ini?

Zooming dan Panning:
Anda bisa melakukan menggeser, memperkecil, dan memperbesar peta interaktif untuk melihat area tertentu sebagai fokus.
Layer Management:
Mampu untuk mengaktifkan dan menonaktifkan berbagai lapisan atau layer data, seperti lapisan jalan, bangunan, dan data demografis.
Pop-ups dan Tooltips:
Informasi yang akan muncul saat Anda menekan atau mengarahkan kursor ke sebuah titik di peta, seperti point lokasi UMKM.

Berikut komponen Web Map dan fungsinya

Komponen Web Map Gambar Fungsi
Tombol "Lihat Peta" Untuk membuka halaman web map atau untuk mengarahkan window menuju Web Map.
Simbol Bangunan Ungu Untuk Menunjukkan Area Bangunan dalam Web Map
Simbol Batas Desa Hijau Untuk Menunjukkan Batas area Desa Sukorejo
Simbol Titik Lokasi UMKM Untuk menunjukkan lokasi UMKM dalam fitur titik yang ketika diklik akan memunculkan informasi tambahan terkait UMKM, seperti nama pemilik, kontak, foto tempat UMKM, dll. Saat pop up muncul, Anda bisa menggeser atau scroll pop up ke atas-bawah atau kiri-kanan untuk menampilkan informasi lebih lanjut.
Tombol Zoom In Untuk mengubah tampilan map menjadi lebih dekat, tetapi cakupan area mengecil/menyempit
Tombol Zoom Out Untuk mengubah tampilan map menjadi lebih jauh, tetapi cakupan area menjadi lebih luas
